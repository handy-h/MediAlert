## 1. 国际化 (i18n) — values-en 完全缺失

### 🔴 Critical — Issue #1: 无英文 fallback，三星 S25 全部显示中文

**现状:**
```
app/src/main/res/
├── values/              ← 全部中文
├── values-night/        ← 仅夜间主题颜色
├── drawable/
├── mipmap-*/
└── xml/
```

- 没有任何 `values-en/` 目录
- `values/strings.xml` 全部为中文字符串
- Android 系统在 Samsung S25（en-US locale）下会直接使用 `values/` 作为 fallback
- **结果：所有 UI 文案、Toast、通知、日历事件标题/描述均为中文**

**修复建议:**
1. 创建 `values/strings.xml` 为英文（设为默认语言）
2. 创建 `values-zh/strings.xml` 为中文
3. 或者：保留现状 `values/` = 中文，但必须创建 `values-en/` 使英文设备至少有英文回退
4. 推荐方式：
```
values/        ← 英文（Google Play 全球最佳实践）
values-zh/     ← 简体中文
values-zh-rTW/ ← 繁体中文（可选，后续）
```

**工作量估计:** 2-4 小时（逐一翻译所有字符串）
